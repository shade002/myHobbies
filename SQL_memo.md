## 基本的なコマンド

* SELECT
* FROM
* WHERE
* GROUP BY …
* HAVING
* ORDER BY
    * ASC …昇順
    * DESC …降順
    ※どちらも指定しなければASC
* LIMIT …どれほどのデータを持ってくるかを指定する。第一引数は取得を開始するデータの位置、第二引数はデータの個数を指定する。
* IS NULL
* NOT NULL
* JOIN ~ ON
* FOREIGN KEY
* REFERENCE
* BETWEEN AND …条件を満たす範囲のデータを引っ張ってくる
    →LIMITは条件ではなく個数で判断してデータを引っ張ってくる点が違う
